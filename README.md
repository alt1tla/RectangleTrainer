# RectangleTrainer
Тренажёр определения периметра и площади прямоугольника с интерактивным графическим интерфейсом и поддержкой командной строки. Проект демонстрирует чистую архитектуру, разделение логики и UI, а также гибкую систему настроек.
## Возможности
- **Два режима работы:** 
  - `Обучение` — сразу показывает решение, формулы и пошаговый расчёт
  - `Проверка знаний` — выводит только статус правильности ответа
- **Два способа задания фигуры:** По сторонам (`A, B`) / По координатам точек (`x1,y1 → x2,y2`)
- **Интерактивный GUI (WinForms):** Клетчатое поле, перетаскивание фигуры мышью, динамический расчёт в реальном времени, мгновенная смена темы (Light/Dark)
- **Консольный режим (CLI):** Полноценная поддержка параметров командной строки, интерактивный цикл ввода, логирование
- **Настройки:** Автосохранение в JSON, выбор сложности, режима, темы, способа измерения
- **Горячие клавиши:** `F1` (справка), `Enter` (проверить), `Ctrl+N` (новое задание), `Ctrl+S` (настройки)
- **Чистая архитектура:** Вся бизнес-логика вынесена в независимую библиотеку `Core`, готова для переиспользования в WPF, MAUI, Avalonia или Unity
## Структура решения
RectangleTrainer/

├── RectangleTrainer.Core      # Библиотека классов: логика, модели, настройки, события

├── RectangleTrainer.Cli       # Консольное приложение

└── RectangleTrainer.WinForms  # Графический интерфейс

## Запуск проекта

### Через командную строку (CLI)
```bash
# Запуск в интерактивном режиме (по умолчанию: testing, sides, difficulty 1)
dotnet run --project RectangleTrainer.Cli
# Режим обучения, измерение по точкам, средняя сложность
dotnet run --project RectangleTrainer.Cli -- --mode learning --measure points --difficulty 2
# Режим проверки, измерение по сторонам, лёгкая сложность
dotnet run --project RectangleTrainer.Cli -- --mode testing --measure sides --difficulty 1
# Показать справку
dotnet run --project RectangleTrainer.Cli -- --help
```

### Через графический интерфейс (WinForms)
1) Откройте решение в Visual Studio
2) Установите проект RectangleTrainer.WinForms запускаемым:
-  ПКМ по проекту → Назначить запускаемым проектом
-   Нажмите F5 (отладка) или Ctrl+F5 (без отладки)
3) Интерактивное поле откроется в центре экрана. Используйте мышь для изменения фигуры, кнопки или горячие клавиши для управления.
